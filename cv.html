Goals

My goal is to grow as a software developer with a strong focus on either backend development (particularly using Java) or systems programming in C/C++.

I am aiming to build a deep understanding of scalable backend systems, efficient architecture design, and low-level software behavior, so I can contribute to performance-critical and well-structured software solutions.

About me

I am an aspiring software developer. My interests include systems programming, backend architecture, and performance-oriented development. I enjoy working with technologies like C/C++, Java, JavaScript, and Qt, focusing not only on using tools but also on understanding how they work internally.

Education

I hold a Bachelor's degree in Software Engineering (specialization 121) from the Sambir Professional College of Economics and Information Technologies, located in Lviv region, Ukraine.

I completed a four-year program focused on software engineering fundamentals, during which I studied a wide range of topics including systems programming, web development, and database design.

Throughout my studies, I gained a solid understanding of how software systems are structured, how applications interact with operating systems, and how data is managed and processed across different layers of a system.

Interests

My main interest is understanding how computers work at different levels — from hardware and operating system behavior to application architecture and data flow between system components.
